# xtoonshared

XToon shader code made based on paper Pascal Barla, Joëlle Thollot, Lee Markosian. X-Toon: An extended toon shader. International Symposium on Non-Photorealistic Animation and
Rendering (NPAR’06), Jun 2006, Annecy, France. 10.1145/1124728.1124749

This shader was made as a Advanced Computer Graphics course assigment at University of Applied Sciences Upper Austria
